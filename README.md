## キッズスクリーンセーバー v1.0

PWA オフライン対応の静的 Web アプリ。タップ・スワイプで 30 種のアニメーションを切り替え、1 秒長押しでお気に入り、2 秒長押しで BGM メニューを表示します。

### 使い方
- `public/index.html` をブラウザで開くと即時利用できます（インストール不要）。
- タップ: 次のスクリーンセーバーへ / スワイプ左右: 前後切替。
- 長押し 1 秒: ★ お気に入り登録・解除。長押し 2 秒: BGM メニュー。
- 右下の薄いボタンから設定画面へ（パスコード初期値: 1234）。
- 設定: 自動切替 10/30/60 秒、BGM/効果音、音量、明るさプリセット、背景黒/白、お気に入りだけ再生。

### PWA
- `public/manifest.json` と `public/sw.js` でオフラインキャッシュ。
- Netlify では `/public` をルートにデプロイ。

### 音
- BGM/効果音は Web Audio API で合成（追加の音源不要）。`public/assets/sounds/` に差し替え可。

### データ保持
- LocalStorage にお気に入り、設定値、最後に見たアニメーションを保存し、次回起動時に復元します。
